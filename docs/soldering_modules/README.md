Перенос компонентов с модулей на основную плату

В этой папке находятся подробные иллюстрации, какие именно элементы нужно выпаять с готовых модулей DC-DC и куда их паять на нашу плату.

### Файлы:

- `xl6009_soldering.png` — Перенос компонентов с модуля XL6009
- `MP2307_soldering.png` — Перенос компонентов с модуля MP2307  
- `black_pill_soldering.png` — Перенос компонентов с Black Pill

Эти изображения сильно упрощают процесс пайки, особенно для новичков.
