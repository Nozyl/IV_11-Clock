# Папка hardware/

Здесь находятся все файлы, связанные с разработкой печатной платы.

### Содержимое

- **`altium/`** — исходный проект Altium Designer (схема + плата)
- **`exported/`** — готовые файлы для просмотра без Altium:
  - `Schematic.pdf` — электрическая схема
  - `Board.pdf` — печатная плата
  - `3d_render.png` — 3D вид платы
- **`gerber/gerber_iv-11_clock.zip`** — файлы для заказа платы на любом производстве
