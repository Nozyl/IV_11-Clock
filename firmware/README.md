# firmware/

Исходный код и готовая прошивка для STM32

### Содержимое
- `Clock_401.bin` — готовая прошивка для STM32F401 (можно сразу заливать)
- `Clock_401.zip` — полный проект для STM32F401 STM32CubeIDE
- `Clock_411.bin` — готовая прошивка для STM32F411 (можно сразу заливать)
- `Clock_411.zip` — полный проект для STM32F411 STM32CubeIDE
